## 使用步骤
##### 1.克隆项目到本地
`git clone https://github.com/Kirito001/vuetify-admin.git`
##### 2.进入刚下载的项目目录
`cd vuetify-admin`
##### 3.安装依赖
`npm install`
##### 4.运行项目
`npm run serve`
## 打包成静态文件（生产环境）
`npm run build`
